# SlayerXPOverlayFabric

A **Fabric mod** for Minecraft that overlays Slayer XP, kills per hour, and quest progress directly on your screen.  

---

## Features

- **Real-time Slayer XP Tracking**  
  Tracks your current Slayer XP for each Slayer boss type and displays it on-screen.

- **Kills per Hour (KPH) Calculation**  
  Automatically calculates your kills per hour.

- **Quest Timer / Stopwatch**  
  Displays a timer for your current Slayer quest to help track efficiency.
  
- **Customizable Overlay**  
  - Move the overlay anywhere on your screen.
  - Toggle individual elements (XP, KPH) on/off.
  - Change colors and formatting to match your UI preferences.

- **Performance Optimized**  
  Lightweight overlay, optimized for minimal impact on FPS.


---

## Installation

1. Install [Fabric Loader](https://fabricmc.net/use/) for **1.21.10** as of right now, we **ONLY** support **1.21.10**.  
2. Install [Fabric API](https://www.curseforge.com/minecraft/mc-mods/fabric-api).  
3. Download the latest release of **SlayerXPOverlayFabric** from [Releases](#).  
4. Place the `.jar` file into your `mods` folder.  
5. Launch Minecraft with Fabric.

---

## Usage
- Use the `/sxp` command for the configuration GUI
